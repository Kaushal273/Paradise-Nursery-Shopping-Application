# Paradise Nursery Application

Paradise Nursery is a React-based web application for browsing and purchasing plants online.  
The application provides users with a simple and interactive interface to explore nursery products, manage a shopping cart, and simulate an online plant shopping experience.

---

## Features

- Browse a collection of indoor and outdoor plants
- Add plants to cart
- Update cart quantity
- Remove items from cart
- Responsive user interface
- React component-based architecture
- Easy-to-maintain project structure

---

## Tech Stack

- JavaScript
- CSS
- React Router (if applicable)
